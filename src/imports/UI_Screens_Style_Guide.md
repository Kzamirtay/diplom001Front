## Экраны приложения

### 1. Экраны авторизации и onboarding

#### Экран входа
| Элемент | Описание |
|---------|----------|
| Поля ввода | Email, пароль |
| Кнопки | "Войти", "Забыли пароль?", "Регистрация" |
| Выбор роли | Родитель / Специалист |

#### Экран регистрации
- Двухэтапная форма: сначала данные родителя/специалиста, затем создание профиля ребенка
- Фото/аватар ребенка (можно выбрать из предустановленных персонажей)

#### Onboarding (первый запуск)
- Приветственная анимация с талисманом приложения
- Краткий интерактивный тур по основным функциям
- Настройка доступности (профиль ОВЗ)

---

### 2. Главный экран (Dashboard)

#### Для ребенка:
| Блок | Описание |
|------|----------|
| Приветствие | Имя и аватар ребенка |
| "Продолжить обучение" | Быстрый доступ к последнему упражнению |
| Карусель | Рекомендуемые упражнения (3-4 карточки) |
| Достижения | Текущий уровень, звезды за сегодня, серия дней |
| CTA | Крупная яркая кнопка "Начать занятие" |

#### Для родителя:
- Список всех детей с аватарами
- Общая статистика по каждому ребенку (прогресс за неделю)
- Уведомления: "Сегодня выполнено 2 упражнения", "Получено достижение"
- Быстрый доступ к чату со специалистом

---

### 3. Каталог упражнений

#### Фильтры (сверху):
- По категориям: Речь / Когниция / Артикуляция
- По сложности: 1-5 звезд
- По времени: 5 мин / 10 мин / 15 мин

#### Карточка упражнения:
| Элемент | Описание |
|---------|----------|
| Иконка | Эмодзи или иллюстрация |
| Название | Крупный шрифт |
| Сложность | Цветные звезды |
| Прогресс | "Пройдено 3/5 уровней" |
| Статус | Замок на недоступных упражнениях |

#### Боковое меню (планшет/десктоп):
- Избранное
- История занятий
- Достижения

---

### 4. Экран упражнения (Игровой)

#### Верхняя панель:
| Элемент | Описание |
|---------|----------|
| Назад | Кнопка с подтверждением выхода |
| Прогресс-бар | Визуальное заполнение или звезды |
| Таймер | Опционально, можно скрыть |

#### Основная область:
- Крупные интерактивные элементы (минимум 80x80px для тач-интерфейса)
- Яркие цвета с высоким контрастом
- Анимации обратной связи:
  - ✅ Правильно: конфетти/звук радости
  - ❌ Неправильно: мягкая подсказка

#### Нижняя панель:
- Подсказка (кнопка с вопросиком)
- Кнопка "Пропустить" (для детей с аутизмом)
- Полоса прокрутки заданий

---

### 5. Экран результатов

#### Поздравительная анимация:
- Талисман приложения аплодирует
- Летящие звезды/конфетти

#### Статистика занятия:
| Показатель | Отображение |
|------------|-------------|
| XP | Крупными цифрами |
| Точность | Процент выполнения |
| Время | Минуты:секунды |
| Уровень | Апгрейд (если есть) |

#### Награды:
- Полученные звезды (бронза/серебро/золото)
- Прогресс бара достижений

#### Действия:
- "Продолжить" — следующее упражнение
- "В меню" — вернуться в каталог
- "Поделиться" — отправить специалисту

---

### 6. Профиль и прогресс

#### Профиль ребенка:
- Аватар (кастомизация: шапки, очки — разблокируются за достижения)
- Имя, возраст, текущий уровень
- Общее количество XP и звезд

#### Вкладка "Навыки":
Радар-диаграмма или столбчатая диаграмма прогресса:
- Фонематическое восприятие
- Внимание
- Память
- Артикуляция
- Словарный запас

#### Вкладка "Достижения":
- Сетка значков (badges)
- Полученные vs. скрытые (??? для мотивации)
- История: "Получено 15.03.2024"

#### Вкладка "История":
- Календарь занятий:
  - 🟢 Зеленый = выполнено
  - 🟡 Желтый = частично
  - ⚪ Серый = пропущено
- Список последних упражнений с результатами

---

### 7. Экраны для родителей

#### Управление профилями:
- Список детей с возможностью добавления нового
- Редактирование данных ребенка
- Настройки доступности (переключение профилей ОВЗ)

#### Статистика и отчеты:
- Графики прогресса за неделю/месяц/все время
- Сравнение по категориям навыков
- Рекомендации системы

#### Назначения от специалиста:
- Список заданий от логопеда
- Дедлайны и приоритеты
- Отметка о выполнении

---

### 8. Чат

#### Список диалогов:
- Аватар специалиста/родителя
- Последнее сообщение
- Индикатор непрочитанных
- Время последнего сообщения

#### Экран переписки:
- Пузыри сообщений (разные цвета для собеседников)
- Быстрые ответы (шаблоны)
- Прикрепление результатов упражнений
- Голосовые сообщения

---

### 9. Настройки

#### Общие:
| Настройка | Опции |
|-----------|-------|
| Язык | Русский / Казахский |
| Размер шрифта | Маленький / Средний / Большой / Очень большой |
| Тема | Светлая / Темная / Высококонтрастная |

#### Доступность (ОВЗ):
| Профиль | Настройки |
|---------|-----------|
| Аутизм-friendly | Спокойные цвета, минимум анимаций |
| СДВГ-режим | Короткие сессии, частые перерывы |
| Звук | Вкл / Выкл / Только вибрация |
| TTS | Скорость речи |

#### Уведомления:
- Напоминания о занятиях (время)
- Достижения ребенка
- Сообщения от специалиста

---

### 10. Админ-панель (для специалистов)

#### Управление пациентами:
- Список детей с фильтрами по диагнозу
- Карточка ребенка с полной статистикой
- Назначение программ и заданий

#### Конструктор упражнений:
- Создание кастомных карточек
- Загрузка аудио/изображений
- Настройка параметров сложности

## Стиль дизайна

### Общая концепция: **"Дружелюбный минимализм"**

Приложение должно быть:
- 🎨 Визуально привлекательным для детей
- 👨‍👩‍👧 Интуитивным для родителей
- 📊 Функциональным для специалистов

## Цветовая палитра

### Основные цвета

| Цвет | HEX | Использование |
|------|-----|---------------|
| Первичный | `#667EEA` | Кнопки действий, акценты |
| Вторичный | `#F093FB` | Фоны, карточки |
| Фон | `#F8F9FE` | Фон приложения |

### Функциональные цвета

| Статус | HEX | Применение |
|--------|-----|------------|
| Успех | `#4ADE80` | Правильный ответ, достижение |
| Ошибка | `#FB7185` | Неправильный ответ (мягкий) |
| Предупреждение | `#FBBF24` | Напоминания, внимание |
| Информация | `#60A5FA` | Подсказки, справка |

### Цвета категорий

| Категория | HEX | Иконка |
|-----------|-----|--------|
| Речевые | `#F59E0B` | 🗣️ Говорящая голова |
| Когнитивные | `#3B82F6` | 🧠 Мозг/пазл |
| Артикуляционные | `#10B981` | 👅 Рот/язык |

## Компоненты

### Кнопки

```
Свойства:
- Border-radius: 12-16px
- Тень: 0 4px 12px rgba(0,0,0,0.1)
- При нажатии: scale(0.95)
- Градиент вместо сплошного цвета
```

### Карточки

```
Свойства:
- Фон: белый
- Border-radius: 16px
- Тень: 0 2px 8px rgba(0,0,0,0.08)
- Hover: подъем на 2px
```

### Иконки

| Параметр | Значение |
|----------|----------|
| Стиль | Outlined или Filled |
| Размер | 24-32px |
| Особенности | Закругленные края |

## Анимации

### Микро-взаимодействия

| Действие | Анимация | Длительность |
|----------|----------|--------------|
| Наведение | Подпрыгивание | 200ms |
| Нажатие | Scale 0.95 | 100ms |
| Успех | Конфетти | 1000ms |
| Ошибка | Тряска | 300ms |

### Переходы между экранами

- Тип: Плавный слайд
- Длительность: 200-300ms
- Easing: ease-in-out

## Резюме

Приложение "НейроРазвитие" использует стиль **"Дружелюбный минимализм"** с:
- 🎨 Яркой, но не агрессивной палитрой
- 😊 Круглыми, мягкими формами
- ✨ Плавными анимациями
- ♿ Полной доступностью
- 🎯 Четкой иерархией информации

Это создает приложение, которое:
- Привлекает детей через игровой подход
- Понятно родителям без инструкций
- Функционально для профессионалов
